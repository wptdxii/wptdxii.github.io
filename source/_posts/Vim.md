# Normal Mode

|Description|Keymap|
|:-|:-|
|Normal mode|Esc / (Ctrl + [) / (Ctrl + c)|
|Left/Up/Down/Right|h / j / k / l|
|Move to the start/end of the line|0 / $|
|Join lines|[count] + J|
|Jump to line|[count] + G|
|Jump to next/previous word||w/b|
